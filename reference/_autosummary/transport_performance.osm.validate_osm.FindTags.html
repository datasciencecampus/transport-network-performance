FindTags

class transport_performance.osm.validate_osm.FindTags(osm_pth, tag_collator: ~transport_performance.osm.validate_osm._TagHandler = <class 'transport_performance.osm.validate_osm._TagHandler'>)[source]

Bases: object

Applies tag collation to OSM file.

Parameters

osm_pthUnion[Path, str]

Path to osm file.

tag_collator_TagHandler, optional

FindTags applies the logic from tag_collator to a pbf file on init, storing the collated tags in a found_tags attribute. Defaults to _TagHandler.

Raises

TypeError :

osm_pth is not of type pathlib.Path or str.

FileNotFoundError :

osm_pth file not found on disk.

ValueError :

osm_pth does not have a .pbf extension.

TypeError :

tag_collator is not an instance of _TagHandler.
